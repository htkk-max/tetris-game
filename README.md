# 俄罗斯方块 (Tetris)

## 游戏玩法说明

### 游戏简介

俄罗斯方块是一款经典的益智休闲游戏，玩家通过控制下落的方块，将其排列成完整的行来消除并获得分数。本项目是使用纯HTML5、CSS3和JavaScript实现的开源教育项目，适合学习游戏开发。

### 基本操作

- **方向键控制**：
  - 左箭头 (←)：向左移动方块
  - 右箭头 (→)：向右移动方块
  - 上箭头 (↑)：旋转方块
  - 下箭头 (↓)：加速下落
  
- **功能键**：
  - 空格键 (Space) 或回车键 (Enter)：开始游戏/暂停游戏/重新开始

- **游戏目标**：尽可能多地消除完整行，获得高分

### 游戏规则

- 方块从顶部随机生成并向下移动
- 玩家可以控制方块的移动和旋转
- 当方块无法继续下落时，会固定在游戏板上
- 当一行被方块完全填满时，该行会被消除
- 消除的行数越多，获得的分数越高
- 随着等级提升，方块下落速度会逐渐加快
- 当方块堆积到顶部时，游戏结束

### 如何开始游戏

1. 在浏览器中打开 `index.html` 文件
2. 等待游戏加载完成
3. 按回车键或空格键开始游戏

### 游戏特色

- 经典的俄罗斯方块游戏体验
- 纯HTML5 Canvas实现，无需安装任何插件
- 清晰的游戏界面和直观的操作
- 实时显示分数、消除行数和当前等级
- 支持PC和移动设备的键盘操作
- 开源教育项目，可下载学习游戏开发

### 技术要求

- 支持HTML5的现代浏览器（如Chrome、Firefox、Safari、Edge）
- 浏览器需启用JavaScript
- 对于移动设备，建议使用支持键盘模拟的浏览器或外接键盘
- 确保设备有足够的性能支持游戏运行
